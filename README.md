# LifeOS

## Descripción

LifeOS es un Sistema Operativo Financiero Personal (Personal Financial Operating System) diseñado para centralizar, organizar y gestionar información financiera personal de manera estructurada.

---

## Arquitectura

El sistema está dividido en los siguientes módulos:

- backend/ → Lógica del sistema
- frontend/ → Interfaz de usuario
- data/ → Almacenamiento de datos
- infra/ → Infraestructura y despliegue
- docs/ → Documentación del proyecto
- scripts/ → Automatización y utilidades
